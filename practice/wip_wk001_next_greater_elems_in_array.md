### Given An Array Find Next Greater Elem of Each Array Item

#### Assumption
```bash
- When Next Greater Elem Is Not Found Then Set -1
```

#### Samples
```bash
Given:      [1, 4, 2,              1,  6,  2, 4]
Output:     [4, 6, pop < pop' ~ 6, 6, push push -1, pop < pop' ~ 4, -1]
            [4, 6, 6,              6, -1,  4, -1]
```
